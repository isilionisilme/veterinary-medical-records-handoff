# scripts

Vista general de carpetas de scripts. Para el detalle de cada script, abre el
`README.md` dentro de la carpeta correspondiente.

| Carpeta | Contenido |
|---|---|
| [`scripts/ci`](ci/README.md) | Preflight local, niveles L1/L2/L3, instalación de hooks y validación de nombre de rama. |
| [`scripts/dev`](dev/README.md) | Arranque y reset de entorno local/Docker, utilidades de depuración y helpers de VS Code. |
| [`scripts/docs`](docs/README.md) | Validaciones de documentación, generación de router docs, clasificación de cambios y sincronización a wiki. |
| [`scripts/quality`](quality/README.md) | Guardrails de calidad visual: brand compliance y design-system compliance. |
